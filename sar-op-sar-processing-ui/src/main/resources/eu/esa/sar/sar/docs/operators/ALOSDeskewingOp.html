ALOS Deskewing

ALOS Deskewing Operator

For ALOS L1.1 data, the annotated times are not zero Doppler times, i.e. the data is distributed in squinted geometry. Therefore, the data generally needs to be deskewed to transfer the data into a zero Doppler like geometry before applying standard SAR processing.

Note: For ALOS-2 products, deskewing is not needed because products are already in zero Doppler geometry.




The following parameter is used by the Operator:

  1. Source Bands: All bands (real or virtual) of the source product. You may select one or more bands. If no bands are selected, then by default all bands will be processed.
