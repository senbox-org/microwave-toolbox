Gabor Filter

Gabor Filter Texture Operator

Convolves the source image with a 2-D Gabor kernel to extract orientation- and scale-tuned texture features. A Gabor kernel is a Gaussian envelope modulated by a complex sinusoid; rotating and re-scaling the kernel produces a bank of filters that respond preferentially to image structure at the chosen orientation, wavelength and aspect ratio. In SAR analysis, Gabor responses are commonly used as input features for classification (urban, agricultural field boundaries, sea-ice texture).

Algorithm

For each source band, the operator builds a single 2-D Gabor kernel with the parameters listed below and convolves it with the band. The output is a real band whose magnitude at each pixel measures the local response of that orientation / scale combination.

Kernel definition (real part):
g(x,y) = exp(-(x'2 + γ2 y'2) / (2 σ2)) · cos(2πx' / λ)
where (x', y') = (x cosθ + y sinθ, −x sinθ + y cosθ).

Input

Any SAR product. Typically run on a calibrated, multilooked, terrain-corrected intensity image (sigma-nought or gamma-nought).

Output

One real-valued response band per selected source band. To build a multi-feature Gabor bank, run the operator multiple times with different theta values and stack the outputs (e.g., 4 orientations × 2 scales = 8 features).

Parameters

  • Source bands: bands to filter.
  • theta: kernel orientation in radians (default 0.6 rad ≈ 34°). Build a multi-orientation feature bank by re-running with theta = 0, π/4, π/2, 3π/4.

The remaining Gabor parameters are fixed in the current implementation: standard deviation σ = 4.0, wavelength λ = 1.0, kernel aspect ratio γ = 2.0, phase offset ψ = 0.3.
