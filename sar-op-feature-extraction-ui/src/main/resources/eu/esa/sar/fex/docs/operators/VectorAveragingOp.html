Vector Averaging (Zonal Statistics)

Vector Averaging Operator

Zonal-statistics reducer for raster + polygon overlay. For every selected source band, computes one of Mean, Min, Max, or Count over the pixels inside each polygon vector, then writes the aggregate back to every pixel of that polygon. Pixels outside every polygon are written as the source band's no-data value.

Algorithm

Two-pass:

  1. Tile-parallel aggregation builds a per-polygon running sum (and count, for Mean) or running extremum (for Min / Max). Polygon-containment is established via SNAP BandMaths virtual-band expressions (see Rasterize).
  2. Per-pixel write substitutes the aggregate.

Inputs

  • A geocoded raster product (geocoding required for polygon containment).
  • One or more polygon vector layers attached to the source product.
  • One or more source bands to aggregate.

Parameters

ParameterNotes
Method Mean | Max | Min | Count
Source bands One target band per source band.
Polygon vectors If empty, all polygon nodes on the source product are used.
