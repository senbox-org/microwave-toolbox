Rasterize

Rasterize Operator

Converts polygon vectors attached to the source product into one or more attribute raster bands. Each selected polygon attribute becomes a new Float64 band where every pixel inside polygon i carries that polygon's attribute value. Pixels outside every polygon are written as the no-data value (-1).

Modes

  • Set to attribute value — the pixel value is parsed directly from the polygon's attribute (numeric string). Non-numeric / empty attributes are written as 0.
  • Binarize — the pixel value is 1 when the attribute string contains the configured target value (case-insensitive), 0 otherwise. Used to convert a category attribute into a per-class mask.

Algorithm

Polygon containment is delegated to the SNAP BandMaths engine via a chained 'polygonName' ? polygonIndex : ... virtual-band expression, the same trick used by Vector Averaging. Per-tile, only the polygons that intersect the tile are included in the expression to keep evaluation fast.
