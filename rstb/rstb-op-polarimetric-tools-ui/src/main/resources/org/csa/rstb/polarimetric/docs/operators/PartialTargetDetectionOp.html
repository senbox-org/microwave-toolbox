Partial Target Detection

Partial Target Detection Operator

Detects partial polarimetric targets — canonically fire scars or burn scars, but also any target whose Pauli-basis scattering mechanism is known — using the geometrical perturbation filter from Marino, Cloude & Woodhouse 2012.

Algorithm

  1. Estimate the target's feature partial scattering vector ttarget from a user-supplied training polygon (supervised mode — the only validated path in this release). For quad-pol the vector is 6-D (C11, C22, C33, C21, C31, C32); for dual-pol it is 3-D (C11, C22, C21).
  2. Normalise to a unit scattering mechanism w = ttarget / ||ttarget||.
  3. Extend w to an orthonormal complex basis via Gram-Schmidt with random complementary vectors. The matrix that maps this basis back to the canonical axes is the transformation matrix U; in the transformed frame the target appears as (1, 0, 0, ...).
  4. For each pixel: build the local covariance matrix C (mean over a square window), convert to its feature partial scattering vector t′, and apply U to get t′·.
  5. Signal-to-Clutter Ratio SCR = |t′·1|2 / Σi>1 |t′·i|2.
  6. Detection mask γ = 1 / sqrt(1 + RedR / SCR), with RedR = SCRtarget (1/threshold2 − 1). The mask is γ when γ ≥ threshold, and 0 otherwise.

Input

  • Quad-pol T3 / C3 covariance product, or dual-pol C2 covariance product (HH-HV / HH-VV / VH-VV / pre-computed C2).
  • One training polygon attached to the source product (Geometry tool in SNAP). Place over a representative fire-scar / partial-target patch.
  • TOPSAR burst-organised products are not supported; deburst first.

Output

  • A single detectionMask band (Unit.COHERENCE, range [0, 1]). 0 = below threshold (no detection); γ ∈ [threshold, 1] = detection strength.

Parameters

ParameterDefaultNotes
Threshold0.98 Detection threshold on γ. Higher = stricter.
Signal to Clutter Ratio (SCR)50.0 Target SCR (sets RedR scaling). Range [2, 100].
Window Size5 Square sliding-window side for covariance averaging.

Reference

Marino, A.; Cloude, S. R.; Woodhouse, I. H. (2012). A Polarimetric Target Detector Using the Huynen Fork. IEEE Transactions on Geoscience and Remote Sensing 50(10), 4039–4047. DOI: 10.1109/TGRS.2012.2189454.
