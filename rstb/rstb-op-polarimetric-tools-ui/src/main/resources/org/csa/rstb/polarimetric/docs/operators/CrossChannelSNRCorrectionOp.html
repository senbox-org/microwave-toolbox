Cross-Channel SNR Correction

Cross-Channel SNR Correction Operator

Estimates and removes the cross-channel noise floor in a quad-polarimetric SAR product by exploiting the reciprocity assumption SHV = SVH. For an ideal monostatic radar this equality holds, so any decorrelation between the HV and VH channels reflects additive noise. The operator uses the local HV-VH coherence inside a sliding window to estimate the channel SNR and rescales the cross-pol channels to remove that noise contribution.

Algorithm (sketch)

Inside each window the operator computes the sample HV-VH coherence:

γcross = | E[SHV · conj(SVH)] | / sqrt(E[|SHV|2] · E[|SVH|2])

From γcross the per-channel noise variance is recovered (assuming the noise is uncorrelated between channels and the signal is reciprocal), and the cross-pol channels are rescaled so their corrected variance equals the noise-free signal variance.

Input

Full quad-pol SLC product. The operator rejects compact-pol or dual-pol inputs.

Output

A new quad-pol product mirroring the input structure, with the HV / VH channels SNR-corrected. The HH and VV channels pass through unchanged.

Parameters

  • Window Size — sliding window size used for the coherence estimate (default 5; range 1–100). Larger windows reduce variance of the coherence estimate at the cost of spatial resolution.
