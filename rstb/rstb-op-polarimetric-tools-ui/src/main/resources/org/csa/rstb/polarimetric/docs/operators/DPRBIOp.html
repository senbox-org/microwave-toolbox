1. DpRBI Formulation

The Dual-pol Radar Built-up Index (DpRBI) is calculated from the processed dual-pol 2x2 wave covariance C2 matrix. It is important to note that hills or mountains facing the radar produce strong backscatter and are often confused with built-up areas. Therefore, we employ a slope layer to mask such areas to reduce false positives. Please include a slope layer along with the C2 matrix (Raster -> DEM Tools -> Compute Slope and Aspect).

DpRBI is computed using the normalized Stokes vector elements. Individually, each normalized Stokes elements characterize different types of built-up areas. Thus, to map the overall built-up areas, we consider a quadratic mean of these normalized descriptors as shown in the figure below and in [1].
