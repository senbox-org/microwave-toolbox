Polarimetric Change Detection (P-CCD / GLRT)

Generalized Likelihood Ratio Test on Polarimetric Covariance Matrices

Detects change between two coregistered polarimetric SAR products by testing the null hypothesis "the two complex Wishart-distributed covariance/coherency matrices are equal" via a generalized likelihood ratio test [1].

For two p×p Hermitian matrices C1, C2 averaged over N looks each:

    ln Q = N · ln |C1| + N · ln |C2| − 2N · ln |(C1 + C2) / 2|
    T    = −2 · ln Q

Under the null hypothesis, T is asymptotically χ2 distributed with p2 degrees of freedom (4 for C2 dual-pol, 9 for T3/C3 quad-pol). The change probability is:

    p_change = Fχ2(p2)(T)

Inputs

  • Reference: a polarimetric C2/C3/T3 covariance/coherency matrix product.
  • Secondary: a coregistered polarimetric product of the same matrix type and dimensions.

Parameters

  • Window Size: square sliding window used to average each input matrix before the test.
  • Number of Looks (ENL): equivalent number of looks per pixel of the input product (~1 for SLC, ~5–10 for multilooked GRD/RTC). Sets the N in the formula above.
  • Significance Level α: false-alarm rate for the binary change mask. Pixel is flagged as changed when p_change ≥ 1−α.

Outputs

  • TestStatistic: T = −2·ln Q.
  • ChangeProbability: p_change ∈ [0, 1] via the Wilson–Hilferty χ2 CDF approximation.
  • ChangeMask: binary 0/1 mask, 1 = changed at the chosen significance level.

Reference

[1] Conradsen, K., Nielsen, A.A., Schou, J., Skriver, H., 2003. A test statistic in the complex Wishart distribution and its application to change detection in polarimetric SAR data. IEEE Transactions on Geoscience and Remote Sensing, 41(1), 4–19.
