Compact-Pol Parameters Operator

Per-pixel compact-polarimetry parameters derived from the C2 covariance matrix of a circular-hybrid (RCH / LCH) acquisition.

Available parameters

  • Stokes intensity (S0) and Stokes S1, S2, S3.
  • Degree of polarisation m = sqrt(S1^2+S2^2+S3^2)/S0.
  • Circular polarisation ratio μC = (S0-S3)/(S0+S3).
  • m-chi, m-delta, m-alpha hybrid decomposition components.
  • Compact-pol H/α (Cloude, Goodenough & Chen 2012).
  • CP Radar Vegetation Index (RVICP).

Inputs

  • Compact-pol C2 covariance product (LCH or RCH circular hybrid).
