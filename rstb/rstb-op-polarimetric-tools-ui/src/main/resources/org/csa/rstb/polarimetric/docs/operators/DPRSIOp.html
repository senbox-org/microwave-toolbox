1. DpRSI Formulation

The Dual-pol Radar Surface Index (DpRSI) is calculated from the processed dual-pol 2x2 wave covariance C2 matrix.

DpRSI is computed using the normalized Stokes vector elements (g1n) and the conventional wave entropy ( Hw ) as shown in the figure below and in [1]-[3].
