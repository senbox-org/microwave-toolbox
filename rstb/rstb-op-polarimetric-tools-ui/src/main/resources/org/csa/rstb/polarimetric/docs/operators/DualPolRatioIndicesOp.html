Dual-pol Ratio Indices (RFDI / Cross-Ratio / Span)

Inputs

A dual-pol C2 covariance matrix product (4 bands: c11, c12_real, c12_imag, c22).

Outputs (selectable)

  • RFDI = (c11 − c22) / (c11 + c22) — Radar Forest Degradation Index. For HH+HV input this is the original Mitchard et al. 2012 definition; for VV+VH input it is the analogous co/cross intensity contrast widely used with Sentinel-1.
  • Cross-Ratio (CR) = c22 / c11. Cross-pol over co-pol intensity. Used in soil moisture and biomass studies.
  • Cross-Ratio (dB) = 10 · log10(c22 / c11).
  • Span = c11 + c22. Total polarimetric power.

Window Size

Optional sliding window (default 1 = no averaging) to smooth the C2 matrix before computing the ratios. Set > 1 for noisier scenes; the existing PolarimetricSpeckleFilter operator is generally preferred.

Reference

Mitchard, E.T.A., Saatchi, S.S., White, L.J.T., et al., 2012. Mapping tropical forest biomass with radar and spaceborne LiDAR in Lopé National Park, Gabon: overcoming problems of high biomass and persistent cloud. Carbon Balance and Management 7, 1.
