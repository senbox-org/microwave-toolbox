Compact Polarimetric Decomposition Operator

   This operator performs the following decompositions on compact pol data:
  • m-chi decomposition
  • m-delta decomposition
  • H-Alpha decomposition
  • 2-layer RVOG decomposition
  • Model-Free 3-Component Decomposition (MF3CC)
   The operator first computes Stokes vector [g0, g1, g2, g3] for each pixel from the averaged covariance matrix, then performs user selected decomposition.

Input and Output

  • The input to this operator should be a simulated compact polarimetric SAR product in complex scattering vector or complex covariance matrix format.
  • The output of this operator is the m-chi, m-delta, H-Alpha or RVOG decomposition result represented by RGB colours.

Parameters Used

   The following processing parameters are needed (see Figure 1):
  • Decomposition: m-chi, m-delta, H-Alpha, RVOG decomposition or Model-Free 3-Component Decomposition
  • Window Size X: X dimension of the sliding window used in computing the mean covariance matrix
  • Window Size Y: Y dimension of the sliding window used in computing the mean covariance matrix
