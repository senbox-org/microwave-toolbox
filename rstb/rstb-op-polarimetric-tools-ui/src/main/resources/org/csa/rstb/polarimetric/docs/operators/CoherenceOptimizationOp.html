PolInSAR Coherence Optimisation

Coherence Optimisation Operator

Polarimetric SAR Interferometry (PolInSAR) coherence optimisation. For a quad-pol coregistered master + N-slave stack the operator recovers, per slave acquisition and per pixel, the three "optimum" interferometric coherences spanning the polarimetric subspace.

Algorithm — per pixel, within a sliding window:

  1. Estimate the Pauli-basis 3×3 coherency matrices T11 (master), T22 (slave), and the cross-Hermitian T12.
  2. Form V1 = T11−1 T12 T22−1 T12H and V2 = T22−1 T12H T11−1 T12.
  3. Solve the general complex eigenvalue problem for both (Jama-based, using the standard 2n×2n real-symmetric embedding).
  4. For each pair of eigenvectors (w1, w2) compute the optimum coherence γopt = (w1H T12 w2) / sqrt(w1H T11 w1 · w2H T22 w2).

Output bands: three coherence magnitudes (coh_opt_1, coh_opt_2, coh_opt_3) or, when Output optimum coherence in complex is enabled, three complex pairs (i_coh_opt_* / q_coh_opt_*) with virtual intensity and phase bands. Date suffix per slave.

Input

  • Full-polarisation coregistered stack with HH, HV, VH, VV bands per acquisition. Master / slave tags follow the SNAP coregistration convention (mst / slv) with date suffixes.

Parameters

ParameterDefaultNotes
Window Size5 Side length of the square sliding window over which T11/T22/T12 are averaged. Must be odd in effect (the operator uses halfWindowSize = windowSize / 2).
Output optimum coherence in complexfalse When true emits complex i/q pairs (plus virtual intensity + phase bands); when false emits coherence magnitudes (|γ|2) only.

Notes & caveats

  • The condition test |det(Treal-embedding)| > 0.001 flags degenerate windows and writes 0 to all outputs. Tune indirectly via Window Size.
  • Output coherence-magnitude bands are squared (|γ|2) by convention, not |γ|.
  • Only quad-pol input is supported. Dual-pol stacks will throw "Full polarisation coregistered stack is expected."

Reference

Cloude, S. R. and Papathanassiou, K. P. (1998). Polarimetric SAR interferometry. IEEE Transactions on Geoscience and Remote Sensing, 36(5), 1551–1565. DOI: 10.1109/36.718859.
