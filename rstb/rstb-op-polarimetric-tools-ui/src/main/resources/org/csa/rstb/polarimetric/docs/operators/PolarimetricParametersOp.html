Polarimetric Parameters

Polarimetric Parameters Operator

Computes a configurable bank of general-purpose polarimetric scalar parameters from a quad-polarimetric or compact-polarimetric SAR product. The operator works on either the original complex scattering matrix or on the locally averaged coherency / covariance matrix (a 3×3 or 4×4 Hermitian matrix produced by spatial multilooking inside a sliding window).

Available outputs

  • Span — total backscattered power, sum of |SHH|2, 2|SHV|2, |SVV|2 for quad-pol.
  • Pedestal Height — ratio of the smallest to the largest eigenvalue of the coherency matrix; a measure of polarimetric noise / depolarisation floor.
  • Radar Vegetation Index (RVI) — ratio of cross-pol to total power, sensitive to volume scattering from canopies.
  • Radar Forest Degradation Index (RFDI) — (HH−HV) / (HH+HV).
  • Canopy Structure Index (CSI), Volume Scattering Index (VSI), Biomass Index (BMI), Interaction Index (ITI) — structural indices from Pope et al. (1994).
  • Ratios: HH/VV, HH/HV, VV/VH (linear power).

Inputs

  • Quad-pol (SHH, SHV, SVH, SVV) SLC product, or
  • Coherency T3 / covariance C3 matrix product (already produced by Polarimetric Matrix Generation), or
  • Compact-pol (RH, RV) hybrid-mode product.

Parameters

  • Use Mean Matrix — when true, parameters are computed on the locally averaged coherency / covariance matrix (recommended). When false, they are computed on the original single-look scattering matrix per pixel (noisier output; only meaningful for fully developed point scatterers).
  • Window Size X / Y — sliding window in range / azimuth used to compute the mean matrix. Default 5×5. Use larger windows for noisier products at the cost of resolution.
  • Output flags — one boolean per output band (Span, Pedestal Height, RVI, RFDI, CSI, VSI, BMI, ITI, HH/VV, HH/HV, VV/VH).

References

  1. Pope, K. O., Rey-Benayas, J. M., Paris, J. F. (1994). Radar remote sensing of forest and wetland ecosystems in the Central American tropics. Remote Sensing of Environment 48(2), 205–219.
  2. Lee, J.-S., Pottier, E. (2009). Polarimetric Radar Imaging: From Basics to Applications. CRC Press. (Chapters on coherency / covariance statistics and polarimetric scalar descriptors.)
