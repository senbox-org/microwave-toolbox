Orientation Angle Correction Operator

If the norm vector of the ground surface is in the incidence plane, then there is no orientation angle shift induced. However, if the surface norm is not in the incidence plane, then orientation angle shift θ is induced. This operator estimates the orientation angle shift θ using the circular polarization method and removes it from the polarimetric data.

Input and Output

  • The input to this operator can be covariance matrix (C3 or C4) or coherency matrix (T3 or T4) generated by Polarimetric Matrix Generation operator.
  • The output of this operator is coherency matrix T3.

Parameters Used

   No processing parameter is required for this operator.
