Import RADARSAT-2 Product

Radarsat-2 is a commercial SAR mission from MDA and CSA.

Operational: 2007-12-14 to present

Frequency Band: C-band

Acquisition Modes: Stripmap, ScanSAR, Spotlight

Product Types:  

  • SLC Single Look Complex
  • SGF Path Image mulitlooked ground range
  • SGX Path Image Plus mulitlooked ground range
  • SSG Map Image mulitlooked ground range projected
  • SCW ScanSAR Wide
  • SCN ScanSAR Narrow

Polarizations:

  • Single Pol (HH, HV, VV, VH),
  • Dual Pol (HH+HV, VV+VH, HH+VV)
  • Full Pol (HH+HV+VV+VH)

Opening products: Products can be opened by selecting the product.xml file. Opening zipped products (*.zip) directly is also supported.

Special Considerations:  NITF format is not supported. GDAL translate can be used to convert.
