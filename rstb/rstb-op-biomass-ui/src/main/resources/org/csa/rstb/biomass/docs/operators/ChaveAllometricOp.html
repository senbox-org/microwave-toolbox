Chave Allometric AGB Operator

Stand-level Above-Ground Biomass from a canopy-height raster (e.g. from RVoG Forest Height, Dual-Pol Forest Height, TanDEM-X or GEDI L2A) via a height-AGB regression. Four preset regressions are available; users can also supply custom (a, b) coefficients.

Presets

PresetFormulaUse case
Saatchi 2011AGB = 2.0 * H1.5pantropical default
Asner 2012AGB = 0.91 * H2.04Amazon basin
Chave 2014AGB = 0.30 * ρ * H2.0uses wood-density band
CustomAGB = a * Hblocal calibration

Per-tree formula (utility)

The static helper chaveEq7AGB(DBH, H, ρ) evaluates Chave et al. 2014 Eq. 7:

AGB[kg] = 0.0673 * (ρ * DBH2 * H)0.976

(For inventory / per-tree workflows; not exposed as a band-level GPF.)

Inputs

  • Canopy height band [m].
  • Optional wood density band [g/cm3] (used by Chave 2014 preset).
  • Optional forest mask band.

Output

  • AGB band, Mg/ha.

Reference

Chave J. et al. (2014). Improved allometric models to estimate the aboveground biomass of tropical trees. Global Change Biology 20(10), 3177-3190. DOI.
