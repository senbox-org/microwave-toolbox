AGB Uncertainty Operator

Analytic error-budget propagation for AGB maps. Combines four uncorrelated error sources in quadrature to produce a per-pixel 1-sigma standard error plus a configurable-z confidence-interval half-width.

Error sources

sigma_AGB / AGB = sqrt( (sensitivity_perDb * sigma_radio_dB)^2
                      + (regression_RMSE / AGB)^2
                      + sigma_allometric_rel^2
                      + sigma_spatial_rel^2 )

Inputs

  • An AGB band [Mg/ha].
  • Optional radiometric SE band [dB] (residual speckle stddev).

Parameters

  • sensitivityPerDb: regression sensitivity. For AGB = a*exp(b*σ0dB), set to b * 10 / ln(10) ≈ 0.23 for the Mitchard 2009 model.
  • regressionRMSE: RMSE of the upstream AGB regression (Mg/ha).
  • allometricRelative: relative uncertainty of the allometric conversion (typ. 0.10-0.20).
  • spatialRelative: relative sampling / multi-look uncertainty (typ. 0.05-0.15).
  • ciZ: z-multiplier for the CI band (1.645 = 90 %, 1.96 = 95 %).

Outputs

  • AGB_SE [Mg/ha], 1-sigma standard error.
  • AGB_CI [Mg/ha], CI half-width = ciZ * SE.

Reference

IPCC Tier-1 forest biomass uncertainty methodology, Scientific Data 13, 107 (2024). DOI. Compatible with CCI Biomass v5 uncertainty layers.
