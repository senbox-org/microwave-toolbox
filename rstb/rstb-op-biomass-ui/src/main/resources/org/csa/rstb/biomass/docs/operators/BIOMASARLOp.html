BIOMASAR-L Operator

L-band (ALOS PALSAR / PALSAR-2 / ALOS-4) variant of the BIOMASAR multi-temporal Growing Stock Volume retrieval. Implemented as a thin preset that forwards parameters to the underlying BIOMASAR operator with L-band-tuned defaults.

Why L-band?

L-band radar saturates higher than C-band in dense canopies (~400 m3/ha vs. ~150 m3/ha), so BIOMASAR-L is preferred for boreal / tropical dense forests. CCI Biomass v5 and JAXA's K&C Initiative both use BIOMASAR-L operationally.
