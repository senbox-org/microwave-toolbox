Single-Pol Coherence Compensation Operator

Compensates single-polarisation interferometric coherence for SNR and system-gain biases. Used as a preprocessing step before RVoG Forest Height or Dual-Pol Forest Height when only one polarimetric channel is available.

Inputs

  • A coregistered single-pol SLC pair (master + slave).
  • Optional NESZ / noise floor metadata (Sentinel-1 CoSSC supplies these).

Outputs

  • SNR-compensated coherence band.

Notes

Uses Sentinel-1 metadata where available (NESZ noise vectors, IW slice parameters). Reverts to a fixed-noise default when the metadata is absent. The companion Dual-Pol Polarimetric Coherence handles the dual-pol variant.
