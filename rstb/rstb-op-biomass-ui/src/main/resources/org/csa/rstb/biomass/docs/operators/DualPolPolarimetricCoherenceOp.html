Dual-Pol Polarimetric Coherence Operator

Computes polarimetric interferometric coherence from a dual-pol coregistered SLC stack, with optional SNR-based compensation for residual speckle / system-noise bias. Companion to Dual-Pol Forest Height Estimation.

Algorithm

  1. Per-pixel C2 (or T2) coherency matrix estimation over a sliding window.
  2. Optimum coherence extraction via eigendecomposition of the dual-pol coherence matrix.
  3. SNR-based bias correction using upstream noise / coherence-window looks metadata.

Inputs

  • Dual-pol coregistered SLC stack (TanDEM-X bistatic, S-1 dual-pol IW).
  • Coherence window parameters (azimuth / range).

Outputs

  • Per-pair coherence magnitude and phase bands.
  • Compensated coherence (when SNR compensation is enabled).

Notes

This is the dual-pol PolInSAR coherence kernel used by the BIOMASS forest-height chain. For quad-pol coherence optimisation see the CoherenceOptimization operator in rstb-op-polarimetric-tools.
