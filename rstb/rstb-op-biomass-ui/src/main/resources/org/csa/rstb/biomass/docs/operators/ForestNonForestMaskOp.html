Forest / Non-Forest Mask Operator

Rule-based binary forest mask from SAR backscatter. v1 cascade:

  1. Cross-pol threshold (VH for S-1, HV for ALOS-2). Pixels below threshold are non-forest.
  2. Optional co-pol / cross-pol ratio test. Excludes urban / dry bare-soil (high ratio) and persistent water (low both).
  3. Optional temporal coefficient-of-variation test. Excludes seasonal croplands (high CV) when annual CV band is supplied.

Inputs

  • Cross-pol backscatter band (VH for Sentinel-1, HV for ALOS-2).
  • Optional co-pol backscatter band (VV / HH).
  • Optional CV band (annual std / mean of multi-temporal stack).

Output

  • forestMask UINT8 band: 1 = forest, 0 = non-forest, 255 = no-data.

Suggested thresholds

  • Sentinel-1 IW: cross-pol > -19 dB, ratio < 12 dB, CV < 0.5.
  • ALOS-2 PALSAR-2: cross-pol > -16 dB, ratio < 15 dB.

Tune for local conditions and validate against ESA WorldCover (2021) or Hansen GFC.

v2 upgrade path

Replace the rule cascade with a Random Forest / SVM classifier on the same feature stack (Cartus 2020 Figure 7) once a training-data ingestion pipeline is wired in.

Reference

Cartus O., Santoro M. (2020). Assessing Forest / Non-Forest Separability Using Sentinel-1 C-Band SAR. Remote Sensing 12(11), 1899. DOI.
