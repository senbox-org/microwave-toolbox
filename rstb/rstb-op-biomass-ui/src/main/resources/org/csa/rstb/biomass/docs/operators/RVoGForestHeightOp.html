RVoG Forest Height Operator

Random-Volume-over-Ground (RVoG) PolInSAR forest height retrieval from a single complex coherence band. v1 uses the zero-extinction SinC simplification:

|gamma_v(hv)| = |sinc(kz * hv / (2 * pi))|

Inverted numerically (bisection) on the principal branch (0, 2 * pi / kz) for the canopy height hv.

Inputs

  • Coherence magnitude band (e.g. coh_opt_1 from CoherenceOptimization, or any pairwise coherence). Treated as the volume coherence after ground compensation.
  • Vertical interferometric wavenumber kz — per-pixel band or scene-constant parameter.

Outputs

  • forestHeight band [m]. Pixels where |gamma| is below minCoh or above maxCoh are masked to no-data.

v2 upgrade path

For full ground-coherence extraction + joint (hv, sigma_extinction) retrieval, use RVoG 3-Stage Inversion. That operator builds on the same SinC kernel exposed by this one (invertHeightSinC(gamma, kz)).

Reference

Cloude S. R., Papathanassiou K. P. (2003). Three-stage inversion process for deriving forest structure from polarimetric SAR interferometry. IEE Proceedings - RSN 150(3), 125-134. DOI.
