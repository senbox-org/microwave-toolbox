BIOMASAR Operator

Retrieves forest Growing Stock Volume (GSV) from a multi-temporal stack of C-band SAR backscatter images via the BIOMASAR algorithm (Santoro et al., Remote Sensing of Environment 130, 39-49, 2013).

For each pixel the operator:

  1. Identifies dense-forest reference pixels in an adaptive window using the Vegetation Continuous Fields (VCF) tree-cover band.
  2. Identifies ground (un-vegetated) reference pixels by inverting the VCF threshold.
  3. Estimates per-image sigma0gr and sigma0df statistics (mean, median, or mode) over the reference sets.
  4. Inverts the Water Cloud Model variant of Santoro et al. for the per-pixel GSV, combining the multi-temporal estimates with sigma0-contrast weights.

Inputs

  • A stack of coregistered C-band sigma0 (or gamma0) backscatter bands (Envisat ASAR, ALOS-1 PALSAR-1 C-band, Sentinel-1).
  • A vegetation mask band.
  • A tree-cover percentage (VCF) band.

Output

  • One GSV band per requested polarisation (m3/ha).

Reference

Santoro M., Cartus O., Mermoz S. et al. (2013). Retrieval of growing stock volume in boreal forest using hyper-temporal series of Envisat ASAR ScanSAR backscatter measurements. Remote Sensing of Environment 130, 39-49. DOI.

See also BIOMASAR-L for the L-band variant.
