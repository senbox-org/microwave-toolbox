Water Cloud Model Decoupling Operator

Attema-Ulaby Water Cloud Model: separates the observed SAR backscatter into a vegetation contribution and a (two-way) attenuated soil contribution, yielding a per-pixel vegetation water content V [kg/m2].

Model

sigma0_obs    = sigma0_veg + tau^2 * sigma0_soil
sigma0_veg    = A * V * cos(theta) * (1 - tau^2)
tau^2         = exp(-2 * B * V / cos(theta))

Inversion

Bisection on V > 0 such that the forward model matches the observed sigma0. Out-of-bracket pixels (snow, urban, water) are written as no-data.

Inputs

  • Observed backscatter band (linear or dB).
  • Optional bare-soil backscatter band (linear) — typically from an upstream Oh / Dubois / IEM model in rstb-op-soil-moisture.
  • Optional per-pixel incidence-angle band (otherwise TPG or constant).

Outputs

  • vegWaterContent [kg/m2]
  • vegBackscatter [linear]
  • soilBackscatterAttenuated [linear]

Reference

Attema E. P. W., Ulaby F. T. (1978). Vegetation Modeled as a Water Cloud. Radio Science 13(2), 357-364. DOI.
