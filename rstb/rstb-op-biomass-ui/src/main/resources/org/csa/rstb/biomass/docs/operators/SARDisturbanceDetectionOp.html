SAR Disturbance Detection Operator

Per-pixel forest-disturbance detection by CUSUM (cumulative-sum) change-point analysis on a multi-temporal SAR backscatter stack. Tuned to detect monotonic backscatter drops typical of deforestation / clear-cut / fire events.

Algorithm

  1. For each pixel, compute baseline mean and standard deviation from the first baselineLength acquisitions.
  2. For each subsequent acquisition, accumulate the one-sided CUSUM statistic S(t) = max(0, S(t-1) + (μ0 - x(t) - k * σ0)).
  3. Flag disturbance the first time S(t) > h * σ0.

Input

  • A coregistered multi-temporal SAR stack with per-acquisition cross-pol backscatter bands. Bands are discovered via StackUtils.getReferenceBandNames + getSecondaryProductNames and sorted chronologically.

Outputs

  • disturbanceMask UINT8: 1=disturbed, 0=undisturbed, 255=no-data.
  • changeIndex INT16: zero-based acquisition index of detected change (-1 if no change).
  • cusumMag FLOAT32: CUSUM statistic at the detection point (confidence proxy).

Parameter tuning

  • baselineLength: 12 (1 year of monthly S-1 IW).
  • cusumK (slack): 0.5 σ0.
  • cusumH (threshold): 5-8 σ0 for ~1% false-alarm rate.

v2 upgrade path

Full BFAST (Verbesselt et al. 2010) — seasonal harmonic + trend + iterative breakpoint search. Tier 3 in the biomass roadmap.
