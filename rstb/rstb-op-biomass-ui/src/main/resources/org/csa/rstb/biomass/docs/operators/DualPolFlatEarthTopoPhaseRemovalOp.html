Dual-Pol Flat-Earth + Topo Phase Removal Operator

Removes the flat-earth phase contribution and the topographic phase (from an external DEM) from a dual-pol interferogram, leaving the deformation / forest-volume phase. Required pre-processing for Dual-Pol Forest Height Estimation.

Algorithm

  1. Compute the orbital flat-earth phase polynomial over the scene (uses jlinda orbit / SLCImage).
  2. Compute the topographic phase by projecting a DEM through the interferometric geometry.
  3. Subtract both from the input interferogram (complex multiplication by exp(-j phi_flat - j phi_topo)).

Inputs

  • Dual-pol coregistered SLC stack with metadata for both master and slave (bistatic mode supported).
  • A DEM band (or use the external DEM dialog).

Outputs

  • Flat-earth + topo-phase corrected interferogram, dual-pol.

References
