Dual-Pol Forest Height Estimation Operator

Forest canopy height retrieval from TanDEM-X bistatic dual-polarisation interferometric coherence. Implements the Chen et al. 2016 inversion: given a coherence band, a per-pixel vertical interferometric wavenumber kz, and optional ground-phase reference, solve for canopy height under the SinC-Random-Volume-over-Ground assumption.

Inputs

  • Dual-pol coregistered TanDEM-X CoSSC product after coherence and flat-earth/topo-phase removal (use Dual-Pol Flat-Earth + Topo Phase Removal upstream).
  • A ground_phase band (or interferogram phase).
  • kz / wavenumber band from interferometric geometry metadata.

Output

  • forestHeight band [m].

v1 limitations and v2 upgrade path

This is the dual-pol (TanDEM-X) variant. For quad-pol (ALOS-2 / BIOMASS) inputs prefer RVoG Forest Height or the full 3-Stage Inversion.

Reference

Chen H. et al. (2016). Forest height inversion of TanDEM-X bistatic SAR data. IEEE TGRS 54(3), 1571-1584. See also the in-source JavaDoc for parameter details.
