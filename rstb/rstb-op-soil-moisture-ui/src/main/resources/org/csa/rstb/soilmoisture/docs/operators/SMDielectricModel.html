SM Dielectric Modeling

Soil moisture plays an important role in the estimation of dielectric properties in soil. By inverting a dielectric model, soil moisture can be retrieved from the soil real dielectric constant (RDC). User can choose one of two models:

  • Hallikainen
  • Mironov

 

  Parameters Used (all models)

All models use the source RDC band.

Outputs (all models)

The soil moisture (SM) is output as a band.

User can select whether to copy the input RDC band to the target product.

User can also select to output the Land Cover bands from the source product to the target product.

 

  User Input (all models)

  User can enter the range of SM (minimum and maximum) for retreival.
 

Hallikainen Model

The Hallikainen model computes soil moisture from RDC plus sand and clay percentages.

Additional Parameters (Hallikainen)

Both sand and clay source bands are used.


  

Mironov Model

The Mironov model computes soil moisture from RDC, clay percentage and effective soil temperature.

Additional Parameters (Mironov)

Clay source band is used.

User Input (Mironov)

User can enter the effective soil temperature.
