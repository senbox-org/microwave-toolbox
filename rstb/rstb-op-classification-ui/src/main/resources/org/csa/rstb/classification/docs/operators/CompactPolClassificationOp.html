Compact-Pol Classification Operator

Unsupervised H/α Wishart classification on a compact-polarimetry C2 covariance product. The compact-pol-aware CP_HAlphaWishartC2 classifier replaces the standard Polarimetric Classification initial-clustering step with a Stokes-vector-based H/α computation appropriate for circular-hybrid acquisitions (RCH / LCH).

Algorithm

  1. Per pixel, compute the 2×2 mean covariance matrix C2 over a sliding window.
  2. Compute the compact-pol Stokes vector and the H/α pair (Cloude, Goodenough & Chen 2012 compact decomposition theory).
  3. Assign initial cluster centres based on the standard 9-zone H/α plane.
  4. Iterate Wishart classification on the C2 matrices for up to maxIterations sweeps.

Input

  • Compact-pol C2 covariance product (LCH or RCH circular hybrid mode).

Output

  • Class index band (UINT8), one of 1-9 zones.

Parameters

ParameterDefaultNotes
Window Size X / Y5Covariance averaging window.
Max Iterations3Wishart re-clustering passes.

Reference

Cloude S. R., Goodenough D. G., Chen H. (2012). Compact Decomposition Theory. IEEE GRSL 9(1), 28-32.
