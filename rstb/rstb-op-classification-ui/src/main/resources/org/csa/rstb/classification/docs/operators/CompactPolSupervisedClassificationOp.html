Compact-Pol Supervised Classification Operator

Supervised Wishart classification for compact-pol products. Reads class-labelled C2 cluster centres from a Java properties training-data file and assigns each pixel to the nearest cluster by Wishart distance, then maps the cluster index to a class index.

Training-data format (Java properties)

number_of_clusters = N
cluster0           = ClassName_0
cluster0_C11       = ...
cluster0_C12_real  = ...
cluster0_C12_imag  = ...
cluster0_C22       = ...
cluster1           = ClassName_1
...

Multiple clusters can share a class prefix (e.g. Forest_0, Forest_1) and are merged at output.

Inputs

  • Compact-pol C2 covariance product.
  • A properties file with labelled cluster centres.

Output

  • supervised_wishart_class UINT8 band, one entry per derived class.

Reference

Lee J.-S., Pottier E. (2009). Polarimetric Radar Imaging: From Basics to Applications (CRC Press, supervised Wishart classifier chapter). Compact-pol specialisation: Cloude, Goodenough & Chen 2012, IEEE GRSL.
