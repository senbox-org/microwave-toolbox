View ASAR External Calibration (XCA) Product

Opens an ENVISAT ASAR XCA (External Calibration) auxiliary file in SNAP for inspection. XCA files are the official ESA-issued instrument calibration data files used by the Radiometric Calibration operator when calibrating ASAR products. They contain antenna pattern look-up tables, calibration constants, and reference elevation profiles, indexed by acquisition mode and swath.

What's in an XCA file

  • Calibration constants — mode- and polarization-dependent gains used to convert raw DN to backscatter (sigma-nought).
  • Antenna elevation patterns — swath-dependent two-way patterns sampled across the elevation angle.
  • Reference incidence angle profile — used by the calibrator to normalize the pattern correction across the swath.
  • Validity period — the start / stop times during which the file is the authoritative calibration reference.

Usage

  1. Select File → Import → SAR Auxiliary Data → ASAR XCA.
  2. Pick the ASA_XCA_* file from disk. The reader exposes the calibration tables as bands and metadata so the user can plot antenna patterns and inspect calibration constants.
  3. For actual calibration of an ASAR scene, use the Radiometric Calibration operator and let it select the matching XCA file from the auxiliary directory automatically; this importer is for inspection / quality-check rather than as a calibration step.

Sources of XCA files

XCA files are part of the ENVISAT ASAR auxiliary data collection at the ESA Earth Online portal (earth.esa.int/eogateway/missions/envisat). SNAP's auxiliary-data manager downloads them on demand the first time a Radiometric Calibration is run on an ASAR product whose acquisition date is not yet covered by a cached XCA file.
