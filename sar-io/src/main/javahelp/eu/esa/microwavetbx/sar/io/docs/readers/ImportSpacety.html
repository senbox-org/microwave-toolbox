Import Spacety

Import Spacety Product

Operator: Spacety (now operating as TY-SAT) — commercial SAR constellation.

Operational: 2020–present.

Frequency Band: C-band.

Acquisition Modes: Stripmap, Sliding Spotlight, Staring Spotlight, ScanSAR.

Product Types:

  • SLC (Single Look Complex)
  • GRD (Ground Range Detected, multilooked)

Polarizations: Single-pol HH or VV; selected modes support dual-pol HH+HV / VV+VH.

Notes: The reader recognizes the manifest XML accompanying each Spacety product delivery and exposes calibration LUTs through the standard SNAP Radiometric Calibration chain (see the SpacetyCalibrator). Orbit information is read from the bundled state vectors; precise orbit replacement requires an external auxiliary file.
