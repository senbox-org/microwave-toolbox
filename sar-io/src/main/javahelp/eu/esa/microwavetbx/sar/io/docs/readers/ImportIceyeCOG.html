Import ICEYE COG

Import ICEYE Cloud-Optimized GeoTIFF Product

Operator: ICEYE Oy (Finland) commercial X-band SAR constellation.

Operational: First spacecraft 2018; constellation now >30 satellites.

Frequency Band: X-band.

Acquisition Modes: Stripmap, Spotlight, Scan, Dwell.

Product Types:

  • SLC, GRD, GEC, GTC delivered as Cloud-Optimized GeoTIFF (COG)
  • Sidecar JSON metadata describing acquisition, geometry, and calibration

Polarizations: Single-pol VV (most modes).

Notes: This is the GDAL-backed reader for the COG-format ICEYE delivery, distinct from the legacy native-HDF5 ICEYE reader (Import ICEYE). The COG format permits partial reads from cloud object storage so SNAP only fetches the window of interest; this reader uses GDAL's VSI streaming layer when the input URL points at S3 / GCS / Azure object storage.
