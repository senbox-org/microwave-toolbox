Import PAZ Product

PAZ is a Spanish dual use Earth observation and reconnaissance satellite developed by Hisdesat. The PAZ satellite uses the same X-band SAR instrument design and data format as TerraSAR-X and TanDEM-X, and operates in the same orbit constellation.

Operational: 2018-02-22 to present

Frequency Band: X-band (9.65 GHz)

Acquisition Modes: Stripmap, ScanSAR, Spotlight, High-Resolution Spotlight, Staring Spotlight

Product Types:

  • SSC Single Look Slant Range Complex (COSAR binary format)
  • MGD Multi Look Ground Range Detected (GeoTIFF format)
  • GEC Geocoded Ellipsoid Corrected (GeoTIFF format)
  • EEC Enhanced Ellipsoid Corrected with terrain correction (GeoTIFF format)

Polarizations: Single Pol (HH or VV), Dual Pol (HH/VV, HH/HV, VV/VH)

Opening products: Products can be opened by selecting the main XML annotation file (e.g., PAZ1_SAR__SSC*.xml). The TSX product format is used, identical to TerraSAR-X products.
