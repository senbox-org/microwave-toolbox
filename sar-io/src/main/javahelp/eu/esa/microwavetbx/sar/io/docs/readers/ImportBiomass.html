Import BIOMASS

Import BIOMASS Product

Operator: European Space Agency (ESA) Earth Explorer 7 mission.

Operational: Launched 29 April 2025.

Frequency Band: P-band (435 MHz, 6 MHz bandwidth) — first space-borne P-band SAR.

Acquisition Modes: Stripmap; Tomography; PolInSAR. Five-year nominal mission, with two interferometric / tomographic phases interleaved with mono-static stripmap mapping.

Polarizations: Quad-pol (HH, HV, VH, VV).

Supported Product Types

The reader auto-detects the product type from the filename prefix (BIO_S = Level 1, BIO_FP = Level 2) and routes to the appropriate parser. Both directory layouts and ZIP archives are accepted.

RFI Annotation Handling

P-band (435 MHz) is heavily contaminated by terrestrial radio-frequency interference. BIOMASS L1 products carry RFI detection / suppression metadata as XML files in the annotation/rfi/ sub-folder. The reader loads every such file under an rfi element in the original metadata so downstream operators can inspect which pulses / bursts were affected and which suppression method was applied.

Similarly, radiometric-calibration auxiliary LUTs from annotation/calibration/ are loaded under calibration, and noise (NESZ) tables from annotation/noise/ are loaded under noise. All three are optional — older products and the simpler DGM type may not carry them, in which case the corresponding metadata element is simply absent.

Notes

This is the GeoTIFF / GDAL-backed reader for BIOMASS products that ship as COG bundles with an XML main product header. Plain ZIP archives are also supported — the reader opens images via GDAL's /vsizip/ virtual filesystem when possible and falls back to temporary extraction otherwise.

See the BIOMASS L2 Reader Spec document (docs/BIOMASS-L2-Reader-Spec.md) in the toolbox repository for the file-layout details and the current implementation status of each L2 phase.
