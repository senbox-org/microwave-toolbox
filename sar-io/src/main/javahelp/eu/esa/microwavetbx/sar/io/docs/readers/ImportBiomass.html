Import BIOMASS Product

Operator: European Space Agency (ESA) Earth Explorer 7 mission.

Operational: Launched 29 April 2025.

Frequency Band: P-band (435 MHz, 6 MHz bandwidth) — first space-borne P-band SAR.

Acquisition Modes: Stripmap; Tomography; PolInSAR. Five-year nominal mission, with two interferometric / tomographic phases interleaved with mono-static stripmap mapping.

Product Types:

  • L1A SCS (Single-Look Slant-range Complex)
  • L1B DGM (Detected Ground Multi-look)
  • L1C SCS_RFI — SLC with RFI suppression applied
  • L2A AGB — Above-Ground Biomass map
  • L2A FH — Forest Height
  • L2A FD — Forest Disturbance

Polarizations: Quad-pol (HH, HV, VH, VV).

Notes: This reader is the GeoTIFF / GDAL-backed variant used for BIOMASS products distributed in geocoded GeoTIFF form. Native L1 SCS products in the official ESA EOF / NetCDF format are also handled by SNAP via the BIOMASS L2 reader; see the "BIOMASS L2 Reader Spec" document in the toolbox docs directory for the file-layout details.
