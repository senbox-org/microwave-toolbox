Import Cosmo-Skymed

Import Cosmo-Skymed Product

  Cosmo-Skymed is a dual use constellation of SAR satellites developed by ASI. Both first generation (CSK, 4 satellites) and second generation (CSG, 2 satellites) products are supported.

Operational: 2007-06-08 to present (CSK), 2022 to present (CSG)

Frequency Band: X-band

Acquisition Modes: Stripmap, ScanSAR Huge, ScanSAR Wide, Spotlight

Product Types:

  • SCS Single Look Complex slant range
  • DGM Multi-look ground range
  • GEC Geocoded Ellipsoid Corrected
  • GTC Geocoded and Terrain Corrected using a DEM

File Formats:

  • HDF5 (.h5) - both CSK and CSG products
  • GeoTIFF (.tif) with XML metadata (.attribs.xml) - CSG products

Polarizations: Single Pol, non-coherent Dual Pol in Ping Pong Stripmap mode

Opening products: Products can be opened by selecting the *.h5 file (HDF5 format), the *.tif file, or the *.attribs.xml file (GeoTIFF format). Opening zipped products (*.zip) directly is also supported.

Special Considerations:  Currently the SCS ScanSAR Huge Region is not supported
