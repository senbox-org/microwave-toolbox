Import RISAT

Import RISAT Product

This reader supports ISRO's RISAT-1 and EOS-04 (RISAT-1A) C-band SAR satellites. Both use the same CEOS data format.

Satellites:

  • RISAT-1 (2012-04-26 to 2017-07-26, decommissioned)
  • EOS-04 / RISAT-1A (2022-02-14 to present, operational)

Frequency Band: C-band (5.35 GHz)

Acquisition Modes: Stripmap, ScanSAR, Spotlight

Product Types:

  • FRS-1 Fine Resolution Stripmap (~3 m resolution, 25 km swath)
  • FRS-2 Fine Resolution Alternate Polarization Stripmap
  • MRS Medium Resolution ScanSAR (~25 m resolution, 115 km swath)
  • CRS Coarse Resolution ScanSAR (~50 m resolution, 223 km swath)
  • HRS High Resolution Spotlight (~1 m resolution, 10 km swath)

Processing Levels:

  • SLC (Single Look Complex) - complex data in slant range
  • GRD (Ground Range Detected) - multi-looked, ground range projected
  • GEC (Geocoded) - terrain-corrected, map projected

File Formats:

  • CEOS format with BAND_META.txt metadata + GeoTIFF imagery
  • CEOS binary format (VOL/LED/IMG/TRL files)

Polarizations: Single Pol, Dual Pol, Full Pol (HH+HV+VV+VH), Compact Pol (RH+RV circular transmit, linear receive)

Opening products: Products can be opened by selecting the BAND_META.txt file, the product directory, or the CEOS volume file (VDF/VOL). ZIP archives are also supported.
