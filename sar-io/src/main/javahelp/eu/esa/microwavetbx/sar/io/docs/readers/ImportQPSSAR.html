Import QPS SAR

Import QPS SAR Product

Reader for SAR products delivered in the QPS (Quad-Polarization SAR / Generic Polarimetric Stack) GeoTIFF interchange format. The reader treats each input file as a tiled GeoTIFF and resolves the polarimetric channel layout (single-pol, dual-pol HH+HV / VV+VH, or full quad-pol) from the sidecar metadata.

Product Types:

  • Coregistered polarimetric stack delivered as multi-band GeoTIFF
  • Calibrated & geocoded backscatter (sigma-nought, gamma-nought)

Notes: This is the GDAL-backed reader; large rasters are read in a windowed / tiled fashion so SNAP does not load the full image into memory. Use Polarimetric Matrix Generation downstream if the source is a quad-pol stack and you need a coherency or covariance matrix output.
