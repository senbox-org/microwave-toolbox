Demodulate Operator

Demodulate (and deramp if applicable) SLC so that its spectrum is in baseband. Any interpolation/resampling can then be performed on the secondary bands of the output product.

Input

  • The input to this operator should be a stack product.

Output

  • The output of the operator is a stack product with each secondary band in baseband. Also, a new band (per secondary, containing the demodulation phase used) is added to the output product.
