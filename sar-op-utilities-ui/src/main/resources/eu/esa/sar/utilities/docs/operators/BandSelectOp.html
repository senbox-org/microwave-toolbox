Band Select

Band Select Operator

The Band Select Operator allows users to create a new product containing only selected bands from the source product. This operator is useful for reducing the size of the dataset by including only the relevant bands.

For Sentinel-1 OCN L1 SLC products, the operator can be used to select specific imagettes. This enables producing an OCN SLC product similar to a stripmap product thereby allowing operators to process the product as a regular stripmap product.

Parameters

  • Polarisations: A list of polarisations to include in the output product.
  • Sub-Images: A list of sub-images or imagettes to include in the output product.
  • Source Bands: A list of source bands to include in the output product.
  • Band Name Pattern: A regular expression pattern to filter bands by name.


Outputs

The operator generates a new product containing the selected bands, along with updated metadata and tie-point grids.

Usage

To use the Band Select Operator, specify the desired parameters (e.g., polarisations, sub-images, source bands) and execute the operator. The resulting product will contain only the selected bands.

Examples

Example usage in a processing graph:

        <node id="BandSelect">
            <operator>BandSelect</operator>
            <parameters>
                <sourceBands>b1,b2</sourceBands>
                <polarisations>VV,VH</polarisations>
                <subImages>WV1_IMG</subImages>
            </parameters>
        </node>
