Sentinel-1 Burst Overlap Double Difference Interferogram

The operator computes the double difference interferogram for co-registered Sentinel-1 TOPS product for overlapped area between adjacent bursts.

Input

  • The input to this operator should be co-registered Sentinel-1 multi-swath SLC product (e.g. IW product with VH and VV polarisations).

Output

  • The output of the operator is the double difference interferogram for all overlapped area between bursts.
  • The coherence can also be output for the same area if selected.

Parameters Used

  • Output Coherence: If this checkbox is selected, coherence band for the overlapped area will be outout.
  • Coherence Window Size: Dimension of the sliding window for coherence calculation.
