ETAD Correction for GRD Product

The operator applies the ETAD correction to Sentinel-1 GRD product. The Extended Timing Annotation Dataset (ETAD) for Copernicus Sentinel-1 is a new auxiliary product developed by ESA (with DLR as contractor), providing users with corrections to improve geometric accuracy of Sentinel-1 SLC images to centimetric levels. The product contains analysis-ready layers for removing the atmospheric path delays, the solid Earth tidal deformation, and the Sentinel-1 system specific effects related to the IPF SAR processor. The current operator ny default applies the Range Correction Sum layer and Azimuth Correction Sum layer to GRD product only. 

Input

  • The input to this operator should be Sentinel-1 GRD product.

Output

  • The output of the operator is the ETAD corrected GRD product.

Parameters Used

  • ETAD File: The ETAD product provided in a SAFE product container.
