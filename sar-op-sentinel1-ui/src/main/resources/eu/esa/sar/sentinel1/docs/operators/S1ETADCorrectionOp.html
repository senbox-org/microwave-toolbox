ETAD Correction for Sentinel-1 Products

The operator applies the ETAD correction to Sentinel-1 TOPS SLC, Stripmap and GRD products. The Extended Timing Annotation Dataset (ETAD) for Copernicus Sentinel-1 is a new auxiliary product developed by ESA (with DLR as contractor), providing users with corrections to improve geometric accuracy of Sentinel-1 Products. For Sentinel-1 TOPS SLC images, the geometric accuracy can be improved to centimetric levels. The product contains analysis-ready layers for removing the atmospheric path delays, the solid Earth tidal deformation, and the Sentinel-1 system specific effects related to the IPF SAR processor. 

The operator currently performs the ETAD correction in two ways:

  1. Compute for each pixel the range and azimuth offsets based on the user selected ETAD correction layers, then resample the image based on the range and azimuth offsets; 
  2. Compute for each pixel the ETAD correction using the phase delay range layers, converts them to phase corrections and saves them as phase correction band. 

The second option above is for TOPS SLC product only and for InSAR application. The phase correction is computed by the following equation: 

PhaseCorrection = -2 * π * f * (troposphericCorrectionRg + geodeticCorrectionRg - ionosphericCorrectionRg + instrumentTimingCalibrationRange)

where f is the radar frequency. With this option, the ETAD correction is converted to phase correction and output as a band. Users can perform the ETAD correction to both the reference and secondary images, then coregister them by running Back Geocoding operator where the phase correction band is also resampled to the reference image grid. Finally users can generate the interferogram by running Interferogram Formation operator where the differential phase correction is computed and applied in the calculation of interferogram.

Input

  • The input to this operator could be the split Sentinel-1 TOPS SLC product, Stripmap SLC product or GRD product.

Output

  • The output of the operator is the ETAD corrected product.

Parameters Used

  • ETAD File: The ETAD product provided in a SAFE product container.
  • Resampling Type: Interpolation methods for resampling the image.
  • Option 1: Checkbox for image resampling selection. 
  • Option 2: Checkbox for output phase correction selection.
