Phase Linking (DS InSAR)

Phase Linking Operator

The Phase Linking operator performs Distributed-Scatterer (DS) InSAR phase estimation over a coregistered SLC stack. For every output pixel it selects a set of Statistically Homogeneous Pixels (SHPs), forms the N×N complex sample covariance over that set, normalises it to a coherence matrix T, and estimates the per-epoch phasor vector that best explains all N(N−1)/2 pairwise phases jointly.

The output is a stack of phase-linked SLC bands — same amplitude as the input, but with phase replaced by the optimal estimate. The resulting stack is a drop-in input to the existing Interferogram / Coherence / Multi-Reference InSAR operators; downstream pairs show much higher coherence over distributed scatterers (vegetation, debris, farmland) than the raw pairwise interferograms.

Input

  • A coregistered SLC stack — output of Back Geocoding + Enhanced Spectral Diversity for TOPS, or Coregistration for Stripmap.
  • Burst-organised TOPS input is not accepted — run TOPSAR-Deburst first.

   Back-Geocoding                        Coregistration
   Enhanced-Spectral-Diversity
   TOPSAR-Deburst              ← required
                              ↓

                              ↓
   2. Interferogram  OR  Multi-Reference InSAR

   7. Range-Doppler-Terrain-Correction  (optional, geocodes the result)

Phase Linking only modifies phase — amplitudes are preserved — so the downstream graph is identical to a regular pairwise-InSAR graph. Skip Phase Linking entirely if you only care about PS-style point targets; the gain is over distributed scatterers (vegetation, debris, soil) where pixel-wise speckle dominates.

Output

  • i_pl_<swath>_<POL>_<date> / q_pl_<swath>_<POL>_<date> — one complex pair per epoch with the phase-linked phase.
  • tempCoh (optional, default on) — Pepe-Lanari goodness-of-fit temporal coherence per pixel, range [0, 1].
  • numSHP (optional, default off) — count of SHP neighbours used per pixel.

Parameters

EstimatorEVD EVD: dominant eigenvector of T (Ferretti SqueeSAR 2011). EMI: lower-bias variant via inverse iteration on |T|−1 ⊙ T (Ansari, De Zan, Bamler 2018).

Temp. Coherence Threshold0.6 Pixels below this are masked out.

v1 Scope and Limitations

  • Single-shot full-stack EVD/EMI. Recommended operating range is N ≤ ~50 epochs. For longer stacks consider subsampling, or wait for the v2 ministack / sequential mode.
  • No iterative MLE refinement (planned for v2).
  • TOPS bursts must be merged upstream (TOPSAR-Deburst).
