StaMPS Export

Use the StaMPS Export to produce data that can be used within the StaMPS applicaiton for Persistent Scattering Interferometry (PSI).

Inputs: 

  1. Coregistered SLC stack with a reference image and at least 4 secondary images.
  2. Interferogram for reference/secondary pairs and an elevation band

Apply precise orbit correction and coregister a stack of SLC products. For S1 TOPS data, use the TOPS backgeocoding to coregister.

Generate interferograms for the stack. For S1 TOPS data, deburst the interferogram product and the stack product.

Add an elevation band to the interferogram product.


The StaMPS Export will take the coregistered stack and interferogram with elevation and produce an output in Gamma format.

You must then run the mt_prep_gamma script from StaMPS to prepare the Gamma formatted data.

Then run stamps in matlab.
